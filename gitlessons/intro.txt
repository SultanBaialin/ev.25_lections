GIT - распределенная система контроля версий 
Это система для отслеживания и ведения истории изменения файлов, в вашем проекте. 

Репозиторий - хранилище вашего кода и истории его изменений.

GitHub - Веб сайт для размещения git- репозиториев в совместной разработке проектов.

Команды GIT:

1.  git init - она создает новый локальный гит репозиторий у вас на пк, в той директории в которой была прописана команда.(Команда вводится один раз в начале)

2. git add - команда для добавления файлов в систему отслеживания гита, после этой команды изменения начинают отслеживаться гитом.

3. git commit -m '<comment>' - как только достигаем определенного момента в разработке, то тогда мы сохраняем и комментируем изменения в локальный резиторий.  

4. git remote add - это команда для того чтобы связать наш локальный репозиторий с удаленным репозиторием, то есь добовляет подключение к репозиторий на гитхабе.
           git remout add <name: origin> <ссылка_на_репо:ssh>
           
5. git push - команда для отправки всех изменений на локальном репозитории в удаленный репозиторий
             git push origin main
